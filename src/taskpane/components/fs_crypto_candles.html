Example Formula

=FS_CryptoCandles(
"BINANCE:BTCUSDT"
, "D"
, "1631022248"
, "09/15/2021"
)

Arguments

symbol
Crypto pair symbol (with exchange name)
resolution
Supported resolution includes
1, 5, 15, 30, 60, D, W, M
from
Interval initial value (can be string of datetime, string of UNIX timestamp or reference to a datetime cell).
to
(optional)
Interval end value (can be string of datetime, string of UNIX timestamp or reference to a datetime cell). If none is provided, the current timestamp is used

Results

Period
List of timestamp for returned candles (displayed in your timezone)
Close
List of close prices for returned candles.
Open
List of open prices for returned candles.
High
List of high prices for returned candles.
Low
List of low prices for returned candles.
Volume
List of volume data for returned candles.
