Example Formula

=FS_EquityFullFinancials(
"AAPL"
,
"bs"
,
"FY"
,
3
)

Arguments

symbol
Stock symbol
statement
Statement can be 1 of these
bs, ic, cf
for Balance Sheet, Income Statement, Cash Flow respectively.
period
(optional)
Period can be 1 of these
TTM, FY, Q, YTD
and their variants. TTM (Trailing Twelve Months) option is available for Income Statement and Cash Flow. YTD (Year To Date) option is only available for Cash Flow. If none is provided, Q is used if statement is bs otherwise TTM will be used.
limit
(optional)
The number of historical periods to be returned. If none is provided, limit is set to 1. If a custom period such as "TTM-2" and "FY2019.Q3" is used, limit will be ignored.
