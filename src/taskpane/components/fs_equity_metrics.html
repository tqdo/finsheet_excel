Example Formula

=FS_EquityMetrics(
"AAPL"
,
"revenue"
,
"FY"
,
3
)

Arguments

symbol
Stock symbol
code
Metric code name
period
(optional)
Period of data (if the chosen metric supports periods, choose among its supported periods and their variants). If none is provided, the default period is used.
limit
(optional)
The number of historical data points to be returned. If none is provided, limit is set to 1. If a custom period such as "TTM-2" and "Q-3" is used, limit will be ignored.
