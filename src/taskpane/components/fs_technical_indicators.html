Example Formula

=FS_TechnicalIndicators(
"OANDA:EUR_USD"
, "D"
, "sma"
, "1631022248"
, "09/15/2021"
, {"timeperiod", 3; "seriestype", "c"}
)

Arguments

symbol
Any symbol
resolution
Supported resolution includes
1, 5, 15, 30, 60, D, W, M
indicator
Indicator code name. Click here for the full list of supported indicators.
from
Interval initial value (can be string of datetime, string of UNIX timestamp or reference to a datetime cell).
to
(optional)
Interval end value (can be string of datetime, string of UNIX timestamp or reference to a datetime cell)
parameters
(optional)
List of all parameters and their value (use , to separate a parameter from its value, use ; to separate parameters). Click here to see which indicators and parameters are supported.
